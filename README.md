# Karawana 🐴

**Dark fantasy trading RPG** — wóz, osiołek i niebezpieczne trakty.

## Koncept

Wcielasz się w woźnicę-handlarza przemierzającego mroczny świat fantasy. Podróżujesz od miasta do miasta, handlując artefaktami, surowcami i informacjami. Każdy trakt to ryzyko — bandyci, potwory, zmieniająca się sytuacja geopolityczna. Rozwijaj swój wóz, ulepszaj ekwipunek i buduj reputację w świecie, gdzie każda decyzja handlowa może zaważyć na losach frakcji.

## Tech stack

- **Silnik:** Godot 4.6 (GDScript)
- **Grafika:** Pixel art 32×32 (LibreSprite / Aseprite)
- **Platforma:** PC + Mac
- **Język:** GDScript (Python-podobny)
